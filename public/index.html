How It Works

Three integrated components working seamlessly together

ESP8266 Sensors

ESP8266 collects data from DHT11 and MQ135 sensors, then transmits it to the server via stable WiFi connection.

Backend Server

Node.js server receives data, stores it in MySQL database, and provides RESTful API for frontend applications.

Web Dashboard

Modern web dashboard displays real-time data, interactive charts, and comprehensive statistics for 24/7 monitoring.

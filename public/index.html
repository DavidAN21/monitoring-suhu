Cara Kerja Sistem

Sistem bekerja melalui tiga komponen utama yang terintegrasi

1

ESP8266 mengumpulkan data dari sensor DHT11 dan MQ135, kemudian mengirimkannya ke server

2

Server Backend

Server Node.js menerima data, menyimpannya di database MySQL, dan menyajikan API untuk frontend

3

Web Dashboard

Dashboard web menampilkan data real-time, grafik history, dan statistik untuk monitoring
